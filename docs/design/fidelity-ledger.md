# Fidelity Ledger

Accepted concept:

- `docs/design/seedbank-insights-concept.png`

Implementation target:

- Electron + React desktop app, 1440 x 1000 primary viewport.

Comparison points to verify before handoff:

1. PSU-inspired palette: navigation chrome uses PSU Green and Forest Green; selected states use Electric Green sparingly.
2. App structure: left navigation, top workbook status, primary Insight Board, and no marketing landing page.
3. Evidence guardrails: visible confidence labels and explicit false-positive/underpowered warnings.
4. Dashboard density: compact scientific panels, charts, queue table, and data-quality panel without nested card stacks.
5. Seed-bank visual atmosphere: lab/seed imagery appears as a supporting banner and does not obscure controls.
6. Typography: control labels, chart labels, table cells, and warnings are legible at desktop size.
7. Core workflow: workbook import changes the dashboard state and deterministic Ask answers cite computed evidence.
